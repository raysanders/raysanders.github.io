About Project Gemela

This project aims to crowdsource Binary Star research, specifically eclipsing binaries.
Gemela is the feminine form of the Spanish word, Gemelo (twin).
Those familiar with the night sky may catch the similarity to Gemini, the twins.

An easy-to-use web interface will help citizen scientists understand how orbital period, temperature, observing angle, and mass ratio affect the light curve of an eclipsing binary star

The app will start users off with a computer generated model (based on the Wilson-Devinney code), using default parameters. Users will then modify the parameters and observe in real-time how the modified parameters affect the stellar pair model.
